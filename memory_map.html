Addr Start Addr End Description
$0000 $1FFF Base ROM

$C000 $DFFF Cartridge RAM.
$E000 $FFFF Base RAM. 8KiB of SRAM is included stock. It is located on the silicon die of the QK-1 SOC itself.

$08 KEYPAD_COL0 Used to scan the keypad Reading: Activates the 74LS245 for reading the keys on column 0. Keys are active low.

$0B KEYPAD_COL3 Used to scan the keypad Reading: Activates the 74LS245 for reading the keys on column 3. Keys are active low.
$0C Piezo Buzzer Makes sounds Any access to this port causes the buzzer to "click", much like the base sound on the Apple 2.
