Memory

Addr Start Addr End Description
$0000 $0fff Base ROM
$1000 $1fff Cartridge address space. Can be rom, ram, whatever! The line "CART_SEL" (active low) is set active when the CPU accesses this part of the memory address. Cartridges may include their own address decoding (which includes IO address space stuff) by "piggybacking" off the cartridge bus (which includes the system bus).
$2000 $3fff Base RAM. 8KiB of SRAM is included stock. It is located on the silicon die of the QK-1 SOC itself.

$00 Keypad Port Drives the rows and columns of the keypad and to read pressed keys. Writing: The two lowest bits select the row lines. Reading: Reads the 8 keys selected by the row selection. All bits are used. (See Figure 0)
